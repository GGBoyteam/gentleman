## 代码组织结构

1. bootstrap 	目录存放程序初始化的代码
2. routes           目录存放我们所有项目的路由文件
3. templates     目录存放前端代码
4. pkg                对第三方库代码封装
5. config           目录存放配置信息，按照单独的逻辑区分单独的配置文件
6. main.go       程序入口
